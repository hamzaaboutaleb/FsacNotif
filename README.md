# FsacNotif

>  This project is still under development.

FsacNotif is a notification management platform designed for the students and administration of the Faculty of Sciences Ain Chock (FSAC).

This project originally started as my bachelor degree project and was first developed using J2EE. Due to time limitations during the academic period, the project was not fully completed.

I still have the original frontend developed with Angular, but unfortunately I lost a large part of the backend code. Currently, I am trying to reconnect everything, understand the original architecture again, and rebuild the backend using Django with a simpler and cleaner modern structure.

There is also a PDF file included in the project that explains the original concept, architecture, and objectives of the application.

I currently work on this project during my free time as both a personal and educational project.

The goal of FsacNotif is to provide a centralized platform for:

- Faculty announcements
- Exam notifications
- Schedule updates
- Event communication
- Student information sharing
